### Featured Project

#### [HEAN](https://github.com/nadirzhon/HEAN-META) — Autonomous Crypto Trading System

A fully autonomous trading engine that combines quantitative strategies with self-evolving AI. Every component — from market data ingestion to order execution to strategy mutation — runs without human intervention.

| | |
|---|---|
| **Architecture** | 12 Python packages, 11 Docker microservices, async event-driven bus |
| **Strategies** | 15 concurrent strategies with multi-agent consensus gating |
| **Performance** | Rust accelerator (60 PyO3 functions, ~100x over pure Python) |
| **Intelligence** | 5-source Hybrid Oracle — Claude, TFT, FinBERT, TCN, Ollama |
| **Autonomy** | Project Prometheus — self-modifying code forge with sandboxed A/B testing |
| **Decision Layer** | Noosphere V2 — multi-agent debate protocol with knowledge graph |
| **Evolution** | Genetic algorithm (CGA) mutates strategy parameters across generations |
| **Interfaces** | Next.js 15 dashboard, SwiftUI iOS app, 30+ REST API endpoints |
| **Testing** | 1,700+ tests across the full stack |
