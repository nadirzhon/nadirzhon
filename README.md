## What I Build

I build **self-evolving autonomous trading systems** at the intersection of quantitative finance, thermodynamic market physics, and frontier AI/ML research.

My flagship project **HEAN** is not a trading bot — it is a financial superintelligence:

- **15 quantitative strategies** — from impulse momentum to funding arbitrage and liquidity cascade hunting
- **5-source hybrid AI oracle** — TCN, Temporal Fusion Transformer, FinBERT, local LLM, Claude Brain
- **Thermodynamic market physics** — temperature, entropy, Szilard engine, phase detection
- **20 frontier AI/ML modules** — Mamba/S4, KAN, DreamerV3, GNN, TDA topology, conformal prediction, flow matching, neuro-symbolic reasoning, and more
- **Project Prometheus** — autonomous self-evolving layer that writes, tests, and deploys its own code
- **Rust PyO3 core** — ~100× speedup on all hot-path calculations
